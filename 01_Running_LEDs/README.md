
# Running LEDs using 8051

## Description
This project demonstrates the Running LED pattern using the 8051 microcontroller. Seven LEDs connected to Port B glow one after another continuously.

## Components Used
- AT89C51 / 8051 Microcontroller
- 7 LEDs
- Proteus 8
- Keil uVision

## Software Used
- Keil uVision
- Proteus 8 Professional

## Files
- ledrun.c → Embedded C source code
- running led.png → Proteus circuit
- running led output.png → Output screenshot

## Working
The program turns ON one LED at a time while turning OFF the previous LED, creating a running LED effect.
